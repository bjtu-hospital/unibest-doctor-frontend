# 患者精确查询接口

**功能**: 根据手机号或姓名精确查询患者信息。

## 接口定义

- **URL**: `/doctor/patients/exact-search`
- **Method**: `GET`

## 请求参数

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `keyword` | string | 是 | 查询关键字（手机号或姓名），必须精确匹配 |

## 响应数据结构

返回类型: `SearchPatientResult`

```typescript
interface SearchPatientResult {
  patients: {
    patient_id: string;
    name: string;
    gender: string;
    age: number;
    phone: string;
  }[];
}
```

## 响应示例

### 1. 成功 - 查询到单人 (手机号或唯一姓名)

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "patients": [
      {
        "patient_id": "P888",
        "name": "精准王",
        "gender": "男",
        "age": 28,
        "phone": "13888888888"
      }
    ]
  }
}
```

### 2. 成功 - 查询到多人 (重名情况)

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "patients": [
      {
        "patient_id": "P999",
        "name": "重名李",
        "gender": "女",
        "age": 22,
        "phone": "13999999991"
      },
      {
        "patient_id": "P998",
        "name": "重名李",
        "gender": "男",
        "age": 45,
        "phone": "13999999992"
      }
    ]
  }
}
```

### 3. 成功 - 未查询到数据

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "patients": []
  }
}
```
